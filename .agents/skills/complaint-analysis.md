# Complaint Analysis Skill

## Purpose

Analyze a citizen complaint using Gemini AI.

---

## Input

- Complaint Title
- Complaint Description

---

## Processing

The AI should:

- Understand the complaint.
- Identify the main issue.
- Determine severity.
- Understand public impact.

---

## Output

- Complaint Summary
- Priority
- Reason
- Tags

---

## Example

Input

"The road near the government school has many potholes causing accidents."

Output

Summary

Road near government school is severely damaged.

Priority

High

Reason

The issue affects public safety and daily transportation.

Tags

Road

Potholes

School

Safety
