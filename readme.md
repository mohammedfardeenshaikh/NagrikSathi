# NagrikSathi

## Overview

NagrikSathi is an AI-powered citizen grievance platform that enables citizens to submit complaints and helps Members of Parliament prioritize constituency development using Google Gemini AI.

---

## Features

- Citizen Complaint Submission
- AI Complaint Analysis
- AI Priority Classification
- Complaint Summarization
- Tag Generation
- MP Dashboard
- Complaint Status Management

---
